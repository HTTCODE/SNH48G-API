# SNH48G-API
收集整理关于SNH48Group的API，包括口袋48，官网，微博等。

## 相关说明

API中**输入参数**使用变量名占位：
```
{name}
```

相关隐私信息中**数字**按位取1：1990 -> 1111；**字符串**统一用'xxxxxx'替换。
